[opendata.by]
Экспериментальная площадка белорусского сообщества открытых данных.
Сайт можно редактировать через Github по адресу https://github.com/opendataby/tools
Рецепт правильной движухи:
  • open source!
  • не забывать людей в титрах своих проектов
  • добавлять инструменты вместе со ссылками на реальный проект, где он был использован
  • при добавлении кидать клич, чтобы собраться и пилить обучалки в виде поста или видео
  • линковать людей, фотки, датасеты, проекты и инструменты

Эксперименты

GPS конвертер и валидатор

Перевод координат из различных GPS форматов в канонический формат с проверкой входных данных на ошибки. tools.opendata.by/gps/

Базовая работа с SVG

Тестирование особенностей встраивания SVG в HTML код для визуализации данных. tools.opendata.by/svg/
